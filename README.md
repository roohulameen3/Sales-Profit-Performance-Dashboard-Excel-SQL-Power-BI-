# Sales & Profit Performance Dashboard (Excel | SQL | Power BI)

This project is an end-to-end **Sales & Profit Performance Dashboard** built using a complete analytics workflow from **Excel → SQL → Power BI**. The main objective is to analyze sales performance, profitability, return rates, and product trends to generate meaningful business insights.

---

## 📌 Project Objective
To track and analyze:
- Region-wise sales and profit performance  
- Monthly sales and profit trends  
- Return rate impact on profitability  
- Top-performing and low-performing products  

---

## 🔄 Workflow
### 1) Excel (Data Cleaning)
- Cleaned raw dataset
- Handled incorrect date formats and missing values
- Prepared structured data for SQL import

### 2) SQL (Data Analysis)
- Imported cleaned data into SQL database
- Created queries/views for:
  - Region performance
  - Monthly trend analysis
  - Return rate calculations
  - Product-level performance

### 3) Power BI (Dashboard & Reporting)
- Imported SQL tables/views into Power BI
- Built interactive dashboard using:
  - KPI Cards
  - Slicers
  - Line charts (monthly trends)
  - Bar charts (region & product performance)
  - DAX measures for calculations

---

## 📊 Dashboard Highlights
- **Total Sales:** $2.27M  
- **Total Profit:** $282.86K  
- **Avg Profit Margin:** 12.06%  
- **Avg Return Rate:** 5.29%  

---

## 💡 Key Business Insight
The **West region** generated the highest sales but also had the highest return rate (**11.8%**), which impacted overall profitability. Reducing returns can significantly improve profit margins.

2. Import cleaned data into SQL database  
3. Run SQL queries/views provided in `SQL_Queries.sql`  
4. Connect SQL database to Power BI
